    - --- Release Notes ------------------------------------------------>

    The unique characters and Seussian settings of the classic 
    children's book are combined with the upcoming live action film to 
    create this action-adventure title from Universal Interactive 
    Studios and Konami. The game allows players to explore magical 
    Who-ville and it's many surrounding Who-land environments.

    Note: This game has been tested to work on Jap and USA DC. However,
          this game might need a PAL fix, so PAL DC owners are invited
          to wait for a PAL fix before burning this game.
