The Windows Server 2003 Active Directory Infrastructure course from LearnKey prepares you with the knowledge and skills needed to plan, implement, and maintain a Windows Server 2003 Active Directory Infrastructure. Expert instructor Dale Brice-Nash demonstrates how to set up and maintain Active Directory structure and topology, and how to plan and implement user, computer, and group security. At the conclusion of the course, you will understand the skills required to manage and maintain an Active Directory infrastructure within a Windows Server 2003 environment, and be prepared to pass MCP exam 70-294.
