CloneCD is a powerful one and innovating engraver of CD Writes in way RAW, which allows you to have a total control on the data that are written.  Therefore, CloneCD produces identical copies of your CD.  In order to be able to make this peculiar type of recording, CloneCD requires a CD recorder that is able to record in this way.  In addition, CloneCD needs to have a cd player that has supported some specific commandos.  With the correct equipment, CloneCD is able to make copies totally functional of any CD.
